# 📚 Smart Learning Platform with AI-Based Recommendation System

## 🔹 Project Overview

This project is a Smart Learning Platform designed to provide personalized learning experiences using an AI-based recommendation system. It helps users discover relevant courses based on their interests and activity.

---

## 🚀 Features

* 👤 User Registration & Login System
* 📚 Course Browsing
* ❤️ Like & Save Courses
* 💬 Comment System
* 🤖 AI-Based Course Recommendation
* 📊 Personalized User Dashboard

---

## 🧠 AI Recommendation Logic

The system analyzes:

* User activity
* Liked courses
* Completed courses

Based on this data, it recommends similar and relevant courses to improve learning efficiency.

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL

---

## ⚙️ How to Run

1. Download or clone the repository
2. Import the database (.sql file) into MySQL
3. Configure database connection
4. Run using XAMPP / Localhost
